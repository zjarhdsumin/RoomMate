# RoomMate
## :star: 프로젝트 설명
HTML, CSS, JAVA, Spring Boot 프레임워크를 사용하여 기숙사 룸메이트를 구할 수 있는 웹 사이트 개발 진행 중
Spring Boot를 경험해보고자 시작하였으며 아직 웹 사이트를 구현 중. 추후 API와 소켓을 통한 채팅 기능을 개발하고자 함
